# slack-battle
A Slack bot to battle against

# Installation 

`npm install`

# Run application

* Set the Slack bot API token in the config `/src/common/config/index.coffee`
* `npm brew-coffee`
* `npm start`

# Tests

* `Test Runner` Mocha
* `Assertions` Chai
* `Mocking` Sinon

To run a test use the command `npm test`
